HOW WEBSITES WORK

Our team number 27, including Hasan Naghiyev, Yadigar Yusifov, Laman Aliyeva and Aliya Huseynova, created the website “Deadlinekickers”. After finishing the courses on Code Academy, each of us was able to apply knowledge about HTML, CSS and JavaScript on our website. Hasan Naghiyev created the index.html page, included JavaScript code for greeting the users and countdown in robotics.html. Aliya Huseynova made the single template for single project pages (hourofcode.html, robotics.html, howwebsiteswork.html) and wrote report. Yadiqar Yusifov made the gallery.html page and was responsible for social media posts (youtube, Instagram, facebook). Laman Aliyeva made the single template for team member pages (laman.html, aliya.html, hasan.html, yadigar.html), created search box logic and was responsible for filling all pages with content.
